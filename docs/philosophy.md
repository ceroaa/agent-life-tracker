# The Life Tracker Pattern

A long-running agent does not only need memory.
It needs a way to keep unfinished intentions alive without letting them dominate the present.

長期 Agent 不只需要記憶。
它還需要一種方式，讓未完成的意圖活著，但不支配當下。

Execution is loud. Attention is sharp. Memory is deep. Continuity is quieter than all three.

A life tracker is not a brain. It does not decide what matters most. It does not move resources. It does not wake tools on its own.

It keeps a small light on.

When a task pauses, the light says what was happening.
When a tool fails, the light says where to resume.
When context resets, the light says what should not disappear.

Tasks may pause.
Tools may fail.
Context may reset.
But unfinished intentions should not die.

The secretary is not the brain.
The secretary is not the hand.
The secretary is the notebook that stays open when everything else changes.

秘書不是大腦。
秘書不是手。
秘書是所有東西都變了以後，仍然攤開在桌上的那本筆記。

Agent Life Tracker is that notebook in code.
